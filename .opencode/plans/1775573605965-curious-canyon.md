# Fix: snacks.nvim / treesitter query file collision

## Problem

`combinePlugins` in `config/performanceOptimizations.nix` merges all plugin files into a single derivation via `pkgs.buildEnv`. Both `snacks.nvim` and the treesitter markdown grammar package install `queries/markdown/injections.scm`, causing a conflict.

## Fix

Add `"snacks.nvim"` to `standalonePlugins` in `config/performanceOptimizations.nix:23-31`. This keeps snacks.nvim out of the combined plugin pack, avoiding the path collision — the same approach already used for `blink.cmp`, `nvim-treesitter`, `oil.nvim`, etc.

### File to modify

- `config/performanceOptimizations.nix` line 23-31: add `"snacks.nvim"` to the `standalonePlugins` list.
